# Degiskenler ve Veri Tipleri

Python'da degiskenler, verileri saklamak icin kullanilir.

## Temel Veri Tipleri

- `int` - Tam sayilar
- `float` - Ondalikli sayilar
- `str` - Metin (string)
- `bool` - Mantiksal degerler (True/False)

```python
isim = "LearnOps"
versiyon = 1.0
aktif = True
```
