# Python'a Giris

Python, ogrenmesi kolay ve guclu bir programlama dilidir. Bu bolumde Python'un temellerini ogreneceksiniz.

## Neden Python?

- Okunabilir ve temiz soz dizimi
- Genis kutuphane ekosistemi
- Topluluk destegi
- Veri bilimi, web gelistirme, otomasyon gibi genis kullanim alanlari
