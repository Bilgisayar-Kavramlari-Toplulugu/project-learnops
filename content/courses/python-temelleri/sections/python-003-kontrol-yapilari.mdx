# Kontrol Yapilari

Python'da program akisini yonetmek icin kontrol yapilari kullanilir.

## if-elif-else

```python
puan = 85

if puan >= 90:
    print("Harika!")
elif puan >= 70:
    print("Basarili")
else:
    print("Tekrar dene")
```

## for Dongusu

```python
for i in range(5):
    print(f"Adim {i + 1}")
```
