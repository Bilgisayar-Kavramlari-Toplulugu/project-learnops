# Git'e Giris

Git, yazilim projelerinde versiyon kontrolu saglamak icin kullanilan dagitik bir sistemdir.

## Git Nedir?

- Dagitik versiyon kontrol sistemi
- Linus Torvalds tarafindan gelistirildi
- Acik kaynak kodlu
- En yaygin kullanilan VCS
