# Dallanma (Branching)

Git'in en guclu ozelliklerinden biri dallanma (branching) sistemidir.

## Branch Islemleri

```bash
git branch feature/yeni-ozellik    # Yeni branch olustur
git checkout feature/yeni-ozellik  # Branch'e gec
git checkout -b feature/hizli      # Olustur ve gec
```

## Merge

```bash
git checkout main
git merge feature/yeni-ozellik
```
