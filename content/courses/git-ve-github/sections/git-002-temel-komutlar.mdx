# Temel Git Komutlari

Git ile calismak icin bilmeniz gereken temel komutlar.

## Baslangic

```bash
git init          # Yeni repo olustur
git clone <url>   # Mevcut repoyu klonla
git status        # Degisiklikleri gor
```

## Degisiklikleri Kaydetme

```bash
git add .              # Tum degisiklikleri stage'e al
git commit -m "mesaj"  # Commit olustur
git push origin main   # Uzak repoya gonder
```
